#  this-is-a-little-surprise
surprise通关流程大纲

第 0 页：假装报错页 (index.html)

一上来先“演戏”，告诉她网络卡了。等 5 秒倒计时一过，文字瞬间变脸，臭屁地展示一下“远峰蓝机主”的身份，带入一点点幽默感。

第 1 页：正式欢迎页 (real-index.html)

干净整洁的 iOS 欢迎大屏，一句话不说先上排面。就是一个简单的入口，让她做好准备进入你的“逻辑世界”。

第 2 页：人机验证页 (verify.html)

结合了她的朋友圈签名。用“海压竹枝”做题，把“竹”藏在马卡龙色的选项里。只有选对了那个代表她名字的字，才能通关，专属感瞬间拉满。

第 3/4 页：反馈页 (success.html / failure.html)

选对了给个大绿勾，选错了给个温柔的报错。这几页节奏很快，就是为了让她觉得这个系统是“活着”的，有反馈。

第 5 页：日期锁定页 (picker.html)

上硬核验证。模拟 iPhone 的日期拨轮，让她选出今天那个重要的日子（农历二月廿六）。选不对？那可进不去最后的庆典。

第 6 页：装傻疑问页 (puzzle.html)

这是一个“情绪缓冲槽”。在谜底揭晓前，发一个思考的表情包，装傻问一句“今天是什么日子呀，好难猜哦”，调皮一下，拉高她的期待值。

第 7 页：高潮庆典页 (birthday.html)

视觉：背景是快速落下的蛋糕雨，很有派对感。
交互：点击一下出一句祝福，最后出现 3-2-1 倒计时。
高潮：屏幕中心出现一根亮着的蜡烛，等她点一下屏幕（模拟吹灭），火苗熄灭冒烟，最后才蹦出“生日快乐，XXX”的大标题。

第 8 页：彩蛋独白页 (egg.html)

放下所有特效，像写备忘录一样真诚地告诉她：这是你第一次做这种项目，虽然可能不完美，但祝她生日快乐的心意最重要。

第 9 页：梗图钝角页 (angle.html)

最后的调侃。用“钝角”这个万能梗做个选择题，让整个流程在轻松愉快的氛围下收尾。
